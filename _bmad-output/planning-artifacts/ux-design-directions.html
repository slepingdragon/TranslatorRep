Caption stack — top to bottom

  • Row 1 — partner speaking, full visibility. Source (Indonesian, dimmer) over target (English, full opacity, larger).
  • Row 2 — speaker self (Bania), peripheral styling. Right-aligned, ≈60% opacity, smaller body size. Reassuring, not performing.
  • Row 3 — TQ-1 particle preservation (loh preserved as "you know"). Target text rendered with +0.4px letter-spacing — the typographic whisper from §"Desired Emotional Response."
  • Row 4 — TranslationUnavailableMarker (state-amber, plain copy). Source still shows; target reads "Translation unavailable." No alarm banner.
  • Row 5 — SundanesePlaceholderRow. Indonesian-script source still visible; target reads [Sundanese] in dim italic. Quiet, not announced. Designed against anti-emotion #2 (othering).

Themes

  • Theme A (Dark): default. surface-base = #0A0A0B; white-on-dark glass.
  • Theme B (Light high-contrast): surface-base = #F5F5F7; dark-on-light glass; state colors deepened (#9C7818 amber, #8B3A36 red) for AA contrast.
  • Theme C (Custom image): user photo + rgba(0,0,0,0.42) overlay + thicker glass blur. Stock gradient here as placeholder.
